# Dashboard com gráficos informacionais de concessionária fictícia

### Tecnologias utilizadas

1. FastAPI para o backend
2. HTML, CSS e JS para o frontend (sem utilização de frameworks como React)
3. PostgreSQL como banco de dados de pessoas a serem cadastradas

### Deploy

O Deploy desta aplicação será feito por meio do Render (servidor para o FastAPI) e da Vercel (hospedar o frontend)
